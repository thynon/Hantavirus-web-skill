免责声明与使用边界

Hantavirus Global Monitor 是一个本地运行的信息监测与研究辅助界面。它旨在帮助用户阅读、组织和追踪 WHO、CDC、PAHO、ECDC 以及国家卫生机构等官方或准官方来源中的公开信息。它不是医疗服务、公共卫生主管机关、实验室诊断系统、紧急预警系统或临床决策工具。

1. 不构成医学建议

本网站展示的任何文字、数字、摘要、风险表述、国家卡片、区域统计或 AI 摘要都不应被理解为医学诊断、治疗建议、预防处方、用药建议、 检测建议、隔离建议或个人健康判断。若你或他人出现发热、肌痛、胃肠道症状、呼吸困难、低血压或其他疑似感染症状，请立即联系合格医疗人员或当地公共卫生机构。

2. 不替代官方发布

本网站会尝试抓取官方公开来源，但可能因网络、接口、CORS、403/429 限制、PDF 解析失败、页面结构变化、语言差异、缓存或本地服务故障而显示延迟、缺失或降级数据。 所有关键结论都应点击原始来源进行核验。公共卫生行动、旅行安排、媒体发布、研究引用和政策判断必须以相关官方机构的原始公告为准。

3. AI 摘要的限制

AI 摘要只应被视为对证据包的辅助整理。系统会限制 AI 输入为来源、URL、发布日期、国家、指标、数值和摘录，并要求结论绑定引用， 但 AI 仍可能产生遗漏、误解上下文、翻译不准确、过度压缩或引用不完整的情况。AI 摘要不得用于推断病例数、死亡数、传播趋势、个人感染概率或未来风险。

4. 数据口径差异

不同机构的数据可能使用不同病例定义、统计周期、地理范围、报告延迟、实验室确认标准、疑似/确诊分类和病种命名。页面会尽量并排展示来源和日期， 但并排展示不代表这些数字可以直接相加、比较或合并。特别是 HPS/HCPS、HFRS、hantavirus infection、Andes virus 等术语在不同地区和报告中可能口径不同。

5. 紧急情况与责任限制

本网站不提供紧急通知服务。如果你需要紧急医疗帮助，请联系当地急救电话或医疗机构。如果你需要公共卫生指引，请联系当地卫生部门。 使用本网站产生的任何行动、延误、误解、引用错误、经济损失、旅行损失、声誉损害或其他直接/间接后果，均应由使用者自行评估并承担风险。

6. 隐私与 API Key

自定义 AI API Key 设计为仅保存在你的本机浏览器 localStorage 中，并在请求摘要时发送到你选择的供应商接口。请勿在共享电脑或不可信环境中保存密钥。 不同供应商会有各自的数据处理、日志、保留和隐私政策；在使用前请阅读对应供应商条款。
